## 🚀 Quick Start

```bash
# Clone and install
git clone https://github.com/devoter-xyz/devoter-landing
cd devoter-landing
pnpm install

# Start development server
pnpm run dev
```

🌐 Visit `http://localhost:3000`

## 🔧 Environment Variables

```.env.local
NEXT_PUBLIC_VOTING_APP_URL=https://vote.devoter.xyz
```
